Dungeon Dice documentation
==========================

Dungeon Dice is a lightweight CLI game where each `enter` command rolls a room event:

- empty room
- treasure room
- trap room
- monster fight

You manage HP, attack, gold, and potions while trying to survive dungeon runs.
